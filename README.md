# 🚀 MT Graduate Career Launchpad

An AI-powered, single-file Streamlit web application designed to support graduates through their career transition. This platform features secure local authentication and three core intelligent career tools powered by advanced AI architecture.

---

## ✨ Features

*   🔐 **Secure Authentication:** File-based user registration and login utilizing SHA-256 password hashing.
*   📝 **AI CV Builder & Tailoring:** Compares your current resume against a target job description to highlight gaps and recommend critical keyword updates.
*   ✉️ **Cover Letter Generator:** Automatically drafts a high-impact, custom cover letter based on your unique career highlights and the target organization.
*   🎙️ **Interactive Interview Coach:** Provides real-time, constructive feedback on your behavioral responses using structured methodologies like the STAR framework.

---

## 🛠️ Tech Stack

*   **Frontend/App Framework:** Streamlit
*   **Language:** Python 3.x
*   **Database/Storage:** JSON (Local file-based system)
*   **Security:** Hashlib (SHA-256 cryptography)
