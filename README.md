# 🚀 MT Career Launchpad

MT Career Launchpad is an AI-powered career development platform built by **MIDDLE TECHNOLOGY** to help graduates and job seekers prepare for employment.

## ✨ Features

- 🤖 AI Resume Analysis (Google Gemini)
- 📄 CV Builder & Optimizer
- 🧠 Career Assessment
- 🗺 Career Roadmap
- 💼 Job Recommendations
- 📚 Learning Hub
- ⚙️ User Settings Dashboard

---

- Google Gemini API

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/PerosHassan/mt-career-launchpad.git
```

Go into the project

```bash
cd mt-career-launchpad
```

Create a `.env` file

```
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

Run Docker

```bash
docker compose up --build
```

Frontend:

```
http://localhost:8501
```

Backend API:

```
http://localhost:8000/docs
```
