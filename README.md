# 🚀 MT Career Launchpad

MT Career Launchpad is an AI-powered career development platform designed to help students, graduates, and job seekers prepare for the modern workforce.

The application combines Artificial Intelligence with career development tools to provide personalized guidance for users.

---

## ✨ Features

- 🏠 Interactive Dashboard
- 🤖 AI Resume Analyzer
- 🧠 Career Assessment
- 📄 CV Builder
- 🗺️ Career Roadmap Generator
- 💼 Job Recommendations
- 📚 Learning Hub
- ⚙️ Settings

---

## 🛠️ Technologies Used

- Python 3.11
- Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/PerosHassan/mt-career-launchpad.git
```

Move into the project folder:

```bash
cd mt-career-launchpad
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add your Google Gemini API key:

```env
GOOGLE_API_KEY=YOUR_API_KEY_HERE
```

> Never commit your real API key to GitHub.

---

## ▶️ Run with Docker

Build and start the application:

```bash
docker compose up --build
```
